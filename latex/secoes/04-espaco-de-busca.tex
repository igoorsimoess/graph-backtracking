\section{Espaço de Busca}

O conjunto de todas as possibilidades de um problema é chamado de \emph{espaço de busca}.

Para visualizar esse conceito, imagine uma árvore:

\begin{itemize}[leftmargin=1.4em]
  \item Cada nó representa uma decisão.
  \item Cada ramo representa uma escolha possível.
\end{itemize}

À medida que avançamos na árvore, novas possibilidades surgem.

\figura{fig:espaco-busca}%
  {Representação de um espaço de busca através de uma árvore de decisões.}%
  {figuras/tikz/espaco-busca}

O papel do algoritmo é percorrer essa árvore até encontrar uma solução válida.
