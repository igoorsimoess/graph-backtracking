\section{Aplicações Práticas}

O Backtracking é utilizado em vários problemas conhecidos. Entre eles:

\begin{itemize}[leftmargin=1.4em]
  \item Sudoku;
  \item Quebra-cabeças lógicos;
  \item Planejamento de rotas;
  \item Escalonamento de tarefas;
  \item Inteligência Artificial;
  \item Jogos;
  \item Problemas envolvendo grafos.
\end{itemize}

Um dos exemplos mais famosos é o Sudoku. Quando uma posição recebe um número inválido, o algoritmo remove essa escolha e tenta outra alternativa. A cada célula preenchida, ele verifica as três regras do jogo (linha, coluna e bloco 3x3); se a verificação falha, o número é descartado na hora, evitando preencher o restante do tabuleiro a partir de uma escolha que já se sabe inválida. É a poda aplicada a um problema do dia a dia.
